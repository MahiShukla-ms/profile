## 👋 About Me

I'm an aspiring **Data Analyst** passionate about transforming raw data into meaningful insights that support business decisions. With hands-on experience in **SQL, Python, Power BI, Excel, Power Query, and DAX**, I enjoy working with large, high-dimensional datasets, uncovering patterns, and building interactive dashboards that communicate key business metrics effectively.

Through academic and personal projects, I've worked on **data cleaning, exploratory data analysis (EDA), feature engineering, KPI reporting, and dashboard development**. I enjoy solving analytical problems using statistical techniques and dimensionality reduction methods such as **Principal Component Analysis (PCA)** to simplify complex datasets while preserving valuable information.

Beyond analytics, I have a growing interest in **Machine Learning** — exploring predictive models, feature selection, model evaluation, and real-world applications of AI. I believe combining data analysis with machine learning helps organizations make smarter, data-driven decisions.

🎯 **Immediate goal:** start my career as an **Entry-Level Data Analyst**, contributing to real business problems while learning from experienced professionals.
🚀 **Long-term goal:** grow into a **Data Engineer** — building scalable data pipelines, designing efficient data architectures, and enabling robust analytics through modern data engineering practices.

I'm always eager to learn new technologies, collaborate with teams, and take on challenging projects that turn data into actionable insights.
